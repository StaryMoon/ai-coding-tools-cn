## 一句话

VS Code 里的开源 autonomous coding agent。

## 最适合

希望在 VS Code 中使用开源 agent，并连接不同模型/API 的用户

## 官方入口

- 官网/主页：https://github.com/cline/cline
- GitHub：https://github.com/cline/cline
- 文档：https://docs.cline.bot/

## 适合选择它，如果

- 你重度使用 VS Code
- 你想自己选择模型 provider
- 你喜欢开源生态

## 不太适合，如果

- 你不想管理 API key 和模型成本
- 你只需要轻量补全

## 优点

- 开源且社区大
- VS Code 内体验自然
- 模型 provider 灵活

## 注意点

- 复杂任务可能消耗较多 token
- 需要理解权限、工具调用和成本

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
