# 维护说明

AI coding 工具更新极快，这个仓库按“信息入口 + 中文选型”维护，不追求复刻每个工具的完整文档。

## 每次更新建议检查

- 官方链接是否可访问
- GitHub repo 是否改名或迁移
- star 快照是否需要更新
- 是否出现新的主流工具
- 工具是否从开源转为闭源或反向变化
- 学生计划、免费额度、价格是否大改
- 文案是否存在过度承诺

## Star 快照

Star 只代表 GitHub 热度，不代表：

- 模型能力
- 代码质量
- 产品体验
- 长期维护质量
- 是否适合你的项目

建议每次大版本更新时刷新 star 快照，不必每天更新。

## 截图和图片

优先使用：

- 自己制作的 SVG 信息图
- 自己截取的本地 demo 图
- 官方明确允许使用的媒体素材

避免：

- 搬运付费教程截图
- 搬运别人公众号/博客图片
- 未注明来源的产品截图拼贴
