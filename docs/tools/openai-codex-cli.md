# OpenAI Codex CLI

> 类型：`terminal-agent`
> 状态：**强烈关注**
> 开源：**是**
> Stars 快照：**94,122**

## 一句话

轻量终端 coding agent，适合 repo 内自动化修改、测试和多工具工作流。

## 最适合

喜欢在终端/本地仓库里让 agent 直接读代码、改代码、跑测试的人

## 官方入口

- 官网/主页：https://github.com/openai/codex
- GitHub：https://github.com/openai/codex
- 文档：https://developers.openai.com/codex

## 适合选择它，如果

- 你已经重度使用 ChatGPT/OpenAI 生态
- 你希望 agent 能直接在本地仓库里工作
- 你喜欢 CLI、脚本化和可审计的修改流程

## 不太适合，如果

- 你只想要传统 IDE 内联补全
- 你的团队不接受终端 agent 工作流

## 优点

- 终端工作流自然
- 适合自动化 repo 任务
- 和 Codex skills / MCP / local tools 组合空间大

## 注意点

- 能力和体验会随 Codex 版本快速变化
- 本地权限、密钥、沙箱配置需要认真管理

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
