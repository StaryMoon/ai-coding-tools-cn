# Roo Code

> 类型：`vscode-agent`
> 状态：**多 agent 编辑器流**
> 开源：**是**
> Stars 快照：**24,288**

## 一句话

VS Code 中的 AI agent 工具，强调不同模式和 agent 协作。

## 最适合

喜欢 VS Code 里多模式、多角色 agent 工作流的用户

## 官方入口

- 官网/主页：https://github.com/RooCodeInc/Roo-Code
- GitHub：https://github.com/RooCodeInc/Roo-Code
- 文档：https://docs.roocode.com/

## 适合选择它，如果

- 你喜欢 Cline 类编辑器 agent
- 你希望使用多角色/多模式工作流
- 你想保持 VS Code 环境

## 不太适合，如果

- 你不想处理复杂配置
- 你只需要简单补全

## 优点

- 开源
- VS Code 内工作流
- 多模式设计

## 注意点

- 需要理解模式、权限和模型成本
- 社区和产品变化较快

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
