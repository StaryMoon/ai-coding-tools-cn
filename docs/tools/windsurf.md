# Windsurf

> 类型：`ai-editor`
> 状态：**AI IDE**
> 开源：**否**
> Stars 快照：**无公开 GitHub star 快照**

## 一句话

AI code editor / agentic IDE 方向的代表之一。

## 最适合

喜欢 AI IDE 形态、希望 agent 和编辑器深度结合的用户

## 官方入口

- 官网/主页：https://windsurf.com/
- 文档：https://docs.windsurf.com/

## 适合选择它，如果

- 你想尝试 Cursor 之外的 AI IDE
- 你偏好产品化体验
- 你愿意接受闭源商业服务

## 不太适合，如果

- 你必须开源自托管
- 你不想切换编辑器

## 优点

- AI IDE 体验
- 适合项目级修改
- 上手门槛低

## 注意点

- 闭源商业产品
- 价格、账号和功能变化较快

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
