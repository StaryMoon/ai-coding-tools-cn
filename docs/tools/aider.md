# Aider

> 类型：`terminal-agent`
> 状态：**经典**
> 开源：**是**
> Stars 快照：**46,778**

## 一句话

经典终端 AI pair programmer，强调和 Git/diff 的协作。

## 最适合

喜欢 Git 驱动、终端 pair programming、清晰 diff 的开发者

## 官方入口

- 官网/主页：https://github.com/Aider-AI/aider
- GitHub：https://github.com/Aider-AI/aider
- 文档：https://aider.chat/

## 适合选择它，如果

- 你喜欢命令行
- 你希望修改和 Git diff 紧密结合
- 你想用多种模型

## 不太适合，如果

- 你需要完整 IDE 体验
- 你不熟悉 Git 工作流

## 优点

- 成熟
- Git 工作流清晰
- 模型支持面广

## 注意点

- UI 不如 AI IDE 直观
- 需要自己管理模型和 key

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
