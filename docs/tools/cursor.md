# Cursor

> 类型：`ai-editor`
> 状态：**主流**
> 开源：**否**
> Stars 快照：**无公开 GitHub star 快照**

## 一句话

AI-first code editor，适合从传统 VS Code 工作流平滑迁移。

## 最适合

想要开箱即用 AI IDE、少折腾 CLI、重视编辑器内体验的人

## 官方入口

- 官网/主页：https://cursor.com/
- 文档：https://docs.cursor.com/

## 适合选择它，如果

- 你希望编辑器里直接完成问答、修改、补全
- 你不想手动组装很多插件
- 你更在意产品体验而不是开源

## 不太适合，如果

- 你必须使用完全开源工具
- 你不想切换编辑器

## 优点

- 上手快
- 产品体验成熟
- 适合普通开发者和学生项目

## 注意点

- 闭源商业产品
- 价格、模型和额度政策变化较快

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
