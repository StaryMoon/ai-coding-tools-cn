# Claude Code

> 类型：`terminal-agent`
> 状态：**强烈关注**
> 开源：**否**
> Stars 快照：**无公开 GitHub star 快照**

## 一句话

Anthropic 官方 coding agent，适合长上下文、多文件修改和复杂任务拆解。

## 最适合

喜欢 Claude 长上下文、希望 agent 深入理解项目并执行多步任务的人

## 官方入口

- 官网/主页：https://docs.anthropic.com/en/docs/claude-code/overview
- 文档：https://docs.anthropic.com/en/docs/claude-code/overview

## 适合选择它，如果

- 你已经使用 Claude 生态
- 你需要长文档/大项目上下文理解
- 你愿意接受商业闭源工具

## 不太适合，如果

- 你只想使用开源本地工具
- 你非常在意 vendor lock-in

## 优点

- 长上下文体验好
- 多步任务规划能力强
- 适合复杂代码解释和重构

## 注意点

- 闭源商业工具
- 账号、地区、订阅和使用限制可能变化

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
