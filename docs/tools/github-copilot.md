# GitHub Copilot

> 类型：`ide-assistant`
> 状态：**主流基线**
> 开源：**否**
> Stars 快照：**无公开 GitHub star 快照**

## 一句话

主流 AI coding 助手基线，适合大多数普通开发者。

## 最适合

希望最稳定、最主流 IDE AI 补全/聊天体验的开发者和学生

## 官方入口

- 官网/主页：https://github.com/features/copilot
- GitHub：https://github.com/microsoft/vscode-copilot-release
- 文档：https://docs.github.com/en/copilot

## 适合选择它，如果

- 你已经用 GitHub/VS Code
- 你需要稳定补全和聊天
- 你有学生权益或公司授权

## 不太适合，如果

- 你必须使用完全开源工具
- 你想要深度 autonomous agent 控制

## 优点

- 主流
- 集成成熟
- 企业和学生路径清晰

## 注意点

- 闭源商业服务
- 不同计划功能差异较大

## 中文使用建议

先用一个小仓库试：让它读 README、修一个小 bug、补一个测试、跑一次 lint。不要一上来就把毕业论文代码或生产仓库交给自动 agent 大改。
